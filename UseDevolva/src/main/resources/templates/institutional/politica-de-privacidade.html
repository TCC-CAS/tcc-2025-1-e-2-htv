Política de Privacidade

1. Introdução

Esta Política de Privacidade descreve como a plataforma Use e Devolva coleta, utiliza e protege os dados pessoais dos usuários, em conformidade com a Lei Geral de Proteção de Dados (LGPD).

2. Dados coletados

Podemos coletar dados como nome, e-mail, telefone, CPF, informações de localização, dados de pagamento e informações necessárias para funcionamento da plataforma.

3. Finalidade do uso dos dados

Os dados coletados são utilizados para permitir o funcionamento da plataforma, realizar transações entre usuários, melhorar a experiência do serviço e garantir segurança nas operações.

4. Compartilhamento de dados

Algumas informações poderão ser compartilhadas com processadores de pagamento, serviços de verificação de identidade ou autoridades quando exigido por lei.

5. Armazenamento e segurança

Adotamos medidas técnicas e organizacionais para proteger os dados pessoais contra acesso não autorizado, perda ou uso indevido.

6. Direitos do usuário

O usuário poderá solicitar acesso, correção ou exclusão de seus dados pessoais, conforme previsto na LGPD.

7. Retenção de dados

Alguns dados podem ser mantidos por período necessário para cumprimento de obrigações legais ou resolução de disputas.

8. Alterações na política

Esta política poderá ser atualizada periodicamente.

9. Contato

Para dúvidas relacionadas à privacidade: suporte@useedevolva.com
